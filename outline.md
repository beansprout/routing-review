# What:
List App for keeping little reference notes for commands, shortcuts, and whatnots.

## Page 1: List Data, Query + Input Data Page

  ### List:

  - displays a table/ list
    - possibly grouped by `ACTION_WHERE` category
    - `user can click on an item and go to detail`
    - `input form, 3 fields`

  ### Query:
  How do I *`inputfield ACTION`* *`input field OBJECT_OF_ACTION`* in *`inputfield ACTION_WHERE`*

  Example
    a	where1		action1		object-of-action1
    b	where2		action2		object-of-action2
    c	where3		action3		object-of-action3
    d	where4		action4		object-of-action4

  ### AddData:
  How to: `inputfield ACTION`
  in: `inputfield ACTION_WHERE`
  to: `inputfield OBJECT_OF_ACTION`
  instructions: `inputfield INSTRUCTION_DATA`
